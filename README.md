# fblog v0.3

## 说明
这个程序是用 beego 框架实现的。

写这个程序，主要是想完成自己的心愿。作为一个计算机从业者，多少想自己实现一个自己的博客。无奈以前水平有限，总是功亏一篑。掐指一算，这应该是我第三次尝试写自己的博客后台。终于算像模像样了。

第一个版本，基本功能都有。不过比较粗糙就是。以后慢慢完善。

## 安装方法
* 默认使用 sqlite 数据库
* 运行服务后，URL /install 执行安装程序。
